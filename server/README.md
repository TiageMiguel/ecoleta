Ecoleta backend its made with Node.js, and written in Typescript.

Built with:

- express
- sqlite
- knex
- multer

First clone this repo, then:

```sh
cd server

# Install the project dependencies.
yarn

# Up database.
yarn knex:migrate

# Seed the database with some values.
yarn knex:seed

# Run the project.
yarn dev
```
