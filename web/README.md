Ecoleta web frontend its made with React.js, and has the purpose to create new 'points' of collecting waste.

- react-dropzone

- react-leaflet

First clone this repo, then:

```sh
cd web

# Install the project dependencies.
yarn

# Run the project.
yarn start
```
